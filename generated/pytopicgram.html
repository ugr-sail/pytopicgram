A Python library for Telegram crawling and topic modeling.

Modules

crawler

This module facilitates the extraction of messages from Telegram channels.

extractor

This module is designed for extracting and processing text data from CSV files and training a BERTopic model on the extracted data.

main

This module is designed to automatically execute the crawler -> preprocessor -> metrics -> extractor -> viewer pipeline

metrics_calculator

This module facilitates the calculation of various metrics for a dataset.

nlp

This module provides functions for natural language processing (NLP) tasks, including reading data from CSV files, applying NLP techniques to text data, and performing sentiment analysis.

preprocessing_extra

This module provides additional preprocessing functions for data manipulation.

preprocessor

This module is designed for preprocessing messages extracted from Telegram channels.

regex_patterns

viewer

This module provides tools for viewing and analyzing the results of a BERTopic model.
