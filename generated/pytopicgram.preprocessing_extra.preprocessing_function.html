pytopicgram.preprocessing_extra.preprocessing_function

pytopicgram.preprocessing_extra.preprocessing_function(df: DataFrame, **kwargs) → None[source]

Perform additional preprocessing on the DataFrame.

This function applies supplementary data cleaning and transformation tasks that are not covered by the main preprocessing functions. It modifies the DataFrame in place.

Current preprocessing steps include:

  • Removing leading and trailing whitespace from specified string columns.

Additional preprocessing steps can be added as needed.

Parameters:

  • df (pd.DataFrame) – The DataFrame to be processed.

  • kwargs – Additional keyword arguments for customizing the preprocessing steps.

Keyword Arguments:

column_list – A list of column names to apply string trimming.
